# Disaster Prediction Model 2025
This repository contains a machine learning pipeline to predict major disaster events.

## Contents:
- disaster_events_model.ipynb: The main Python notebook containing data preprocessing, PCA feature engineering, and a Random Forest Classifier.

- unclean_dataset.csv: The raw dataset used to train the model.

## How to Run code:
- click the "disaster_events_model.ipynb" file
- then click the "Open in Colab" button on the top right of the preview page
- you are redirected to google colab, and simply click run all
